Sobre FacturaView

Visualiza, entiende y exporta tus facturas electrónicas Facturae de forma gratuita y privada.

¿Qué es FacturaView?

FacturaView es una herramienta web gratuita diseñada para autónomos y pymes españoles que necesitan visualizar facturas electrónicas en formato Facturae (XML).

Con FacturaView puedes abrir cualquier factura Facturae y ver toda su información de forma clara: emisor, receptor, líneas de detalle, impuestos y totales. También puedes exportar la factura a PDF o Excel para archivarla o compartirla.

¿Por qué es gratis?

FacturaView nació de una necesidad real: muchos autónomos y pequeñas empresas reciben facturas electrónicas de la Administración Pública pero no tienen herramientas sencillas para abrirlas.

Las alternativas existentes suelen requerir:

  • Instalar software de escritorio
  • Tener Java instalado
  • Pagar por aplicaciones de facturación completas
  • Subir tus facturas a servidores de terceros

FacturaView elimina todas estas barreras: funciona directamente en el navegador, sin instalación, y procesa las facturas localmente en tu dispositivo.

Privacidad: procesamiento local

Tus facturas nunca salen de tu dispositivo*

El parser XML de FacturaView se ejecuta 100% en tu navegador. Cuando subes un archivo:

  1. El archivo se lee localmente con JavaScript
  2. El XML se parsea en tu navegador
  3. Los datos se muestran en pantalla
  4. Los exports (PDF/Excel) se generan localmente

* La única excepción es la verificación de firmas digitales, que requiere librerías criptográficas no disponibles en el navegador. Si tu factura está firmada y quieres verificar la firma, el XML se envía al servidor para la validación. Esta verificación es opcional.

Funcionalidades

Visualización

  • Soporta Facturae 3.2, 3.2.1 y 3.2.2
  • Muestra emisor y receptor
  • Tabla de líneas de detalle
  • Desglose de impuestos (IVA, IRPF)
  • Información de pago (IBAN, vencimiento)

Exportación

  • Exportar a PDF legible
  • Exportar a Excel (3 hojas)
  • Generación 100% local
  • Sin marcas de agua

Verificación de firma

  • Detecta firmas XAdES
  • Verifica integridad de la firma
  • Muestra datos del firmante
  • Identifica nivel de firma

Historial local

  • Guarda facturas en el navegador
  • Acceso rápido a facturas recientes
  • Datos nunca salen del dispositivo
  • Control total sobre tus datos

Limitaciones

FacturaView es un visualizador, no un programa de facturación completo. Algunas cosas que no hace:

  • No crea facturas: Solo visualiza facturas existentes
  • No firma facturas: Necesitas un programa de facturación para firmar
  • No envía a FACe: El envío lo debes hacer tú desde face.gob.es
  • Validación técnica, no oficial: La verificación de firma no tiene efectos legales. Para validación oficial usa VALIDe

Tecnología

FacturaView está construido con tecnologías web modernas:

  • Frontend: JavaScript vanilla (ES Modules), Vite, Tailwind CSS
  • Backend: Python (FastAPI) solo para validación de firmas
  • PDF: jsPDF (generación 100% cliente)
  • Excel: SheetJS (generación 100% cliente)
  • Hosting: Railway
